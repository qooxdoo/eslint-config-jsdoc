# eslint-config-jsdoc

a disable configuration for jsdoc

## Install

```
$ npm install --save @qooxdoo/eslint-config-jsdoc
```


## Usage

```json
{
	"extends": "@qooxdoo/eslint-jsdoc-config"
}
```
